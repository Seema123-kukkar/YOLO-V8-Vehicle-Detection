🚀 Real-Time Object Detection using YOLOv8 (Images & Videos)
📌 Project Overview

This project implements an end-to-end Object Detection System using YOLOv8 by Ultralytics.

The system is capable of:

✅ Training a custom object detection model

✅ Evaluating performance using standard detection metrics

✅ Running inference on images, videos, and webcam

✅ Saving annotated outputs

✅ Generating performance reports



🧠 Skills Gained From This Project

Computer Vision fundamentals

Bounding Boxes

IoU (Intersection over Union)

Confidence Scores

Training & using YOLOv8 models

Dataset preparation in YOLO format

Model evaluation (mAP, Precision, Recall)

Error analysis (False Positives / False Negatives)

Image, video & webcam inference

Basic deployment mindset (Streamlit/Flask optional)

This project demonstrates a complete real-time object detection workflow using YOLOv8 — from dataset preparation to deployment-ready inference.
